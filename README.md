## 简介

a small tool library.

## Masotool名称的由来

Masotool = March April September October 的首字母 + tool，它是一个轻巧的工具库。

## 安装

在项目的pom.xml的dependencies中加入以下内容:

```xml
<dependency>
    <groupId>io.github.maso-os</groupId>
    <artifactId>masotool-all</artifactId>
    <version>0.1.0</version>
</dependency>
```
